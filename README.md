### 1. 安装 Cemu - Wii U 模拟器

[Cemu ](https://github.com/cemu-project/Cemu) 是一颗 Wii U 模拟器，能够运行大多数Wii U游戏。

- 安装

  你可以从 [Cemu Release](https://github.com/cemu-project/Cemu/releases) 下载最新的应用程序包并解压到合适的位置，比如 `D:\Games\Cemu\`。
